大作业（游戏开发：雷电） 代码文件使用指南

2017级ACM班程序设计B班助教

文件介绍

注：此部分由于代码文件结构调整，有所变化，请参考新版代码文件附带的readme.md。

  • main.cpp： 你需要完成这个文件中的函数来进行游戏开发，这个文件初始时是一个简单的 demo。
  • main2.cpp： 这个文件中是一个较为复杂的 demo，几乎演示了头文件中所有功能的使用。
  • SDL2_header.h： 提供SDL2封装接口的头文件。
  • pointd.hpp： 提供了一个已实现好的Point类，可用于游戏中的几何计算。
  • res_path.h和cleanup.h： 封装所需头文件，你不需要关心其中的内容。

只需要在main.cpp中包含SDL2_header.h即可。

你需要在main.cpp中完成的函数

initialize

void initialize();

程序开始时将会被调用的函数。

work

int work( bool& stop );

主循环中每次将此函数调用一次来实现一帧的刷新，将stop赋值为true将会结束主循环，
程序的返回值为本函数最后一次的返回值。

mousePress

void mousePress();

用户点击鼠标时将会调用此函数。

mouseMove

void mouseMove();

用户移动鼠标时将会调用此函数。

mouseRelease

void mouseRelease();

用户松开被按压的鼠标按键后将会调用此函数。

keyDown

void keyDown();

用户按下键盘按键时将会调用次函数。

keyUp

void keyUp();

用户放开被按下的键盘按键时将会调用此函数。

finale

void finale();

程序结束时将会调用此函数。

数据类型

Image

保存图片的类型
以指针类型保存，最后需要使用cleanup()函数释放（不要用delete）

Color

保存颜色的类型

成员

* r
    红色的深度(0~255)
* g
    绿色的深度(0~255)
* b
    蓝色的深度(0~255)
* a
    不透明度(0~255)

Rect

矩阵类型

成员

* x (int)
    横坐标
* y (int)
    纵坐标
* w (int)
    矩阵宽度
* h (int)
    矩阵高度

Point

保存一个点的类型

成员

* x (int)
    横坐标
* y (int)
    纵坐标

PointD

用于计算几何的点类型（也可用于存储向量）

成员

* x(double)    
    横坐标
* y(double)
    纵坐标
* double length()
    向量长度
* void rotate(double rad)
    以原点为中心旋转rad角度

相关函数

* +,-
    向量的加减运算，返回值为 PointD
* *,/
    向量与数的乘法、除法，返回值为 PointD
* double cross(const PointD &a, const PointD &b)
    向量叉乘
* double dot(const PointD &a, const PointD &b)
    向量点乘

FlipType

图片绘制时是否需要翻转（含义见函数drawImage）

取值

* SDL_FLIP_NONE
    不翻转
* SDL_FLIP_HORIZONTAL
    水平翻转
* SDL_FLIP_VERTICAL
    上下翻转

变量

FPS_DISPLAY

bool 是否在画面中显示FPS(每秒帧数)的值

nowFPS

double 游戏当前的FPS

duration

double 程序目前运行的总时长

mousePressed

bool 鼠标是否被按下

mouseDragged

bool 鼠标是否被拖动

mouseButton

uint8_t 被按下的鼠标按键

mouseX

int 鼠标位置的x坐标

mouseY

int 鼠标位置的y坐标

pmouseX

int 鼠标上一位置的x坐标

pmouseY

int 鼠标上一位置的y坐标

keyPressed

bool 键盘是否被按下

keyValue

int 被按下的按键

canvasColor

Color 画布的背景颜色

SCREEN_WIDTH

int 游戏画面的宽度（需要在main.cpp中定义）

SCREEN_HEIGHT

int 游戏画面的高度（需要在main.cpp中定义）

可供使用的函数

textToImage

Image* textToImage( const std::string &msg,
                    const int32_t &size = fontSize, const Color &color = {255, 255, 255, 255},
                    const std::string &fontType = fontName)

将文本转换为图片，用于在画面上显示文字。

参数

* msg
    需要转换的文本内容
* fontSize
    字体大小
* color
    文本颜色
* fontType
    字体

loadImage

Image* loadImage( const std::string &file )

加载图片文件。

参数

* file
    图片文件名

setImageAlpha

void setImageAlpha( Image *img, Uint8 alpha )

设置图片不透明度。

参数

* img
    图片的指针
* alpha
    不透明度

drawImage

void drawImage(Image *img, int x, int y,
               const double &widthRate = 1,    const double &heightRate = 1,
               const double &angle = 0, const Point *center = NULL,
               const FlipType &flip = FLIP_NONE,
               const Rect *clip = nullptr)

将图片绘制到画面上。

参数

* img
    图片的指针
* x, y
    绘制的图片左上角在画面的位置坐标
* widthRate, heightRate
    绘制时图片宽度和高度的缩放比例
* angle
    绘制时图片的旋转比例，采用角度单位
* center
    旋转中心，若为默认值NULL，则以图片中心为旋转中心
* filp
    图片翻转方式
* clip
    若要绘制图片的部分内容，此参数为一个矩形，描述要绘制的图片部分

getImageSize

void getImageSize( Image *img, int &width, int &height )

获取图片大小。

参数

* img
    图片的指针
* width
    将获取的图片宽度存放在此变量中
* height
    将获取的图片高度存放在此变量中
